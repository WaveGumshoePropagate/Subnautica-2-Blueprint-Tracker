# Installation

## Requirements

* Windows 10 or Windows 11
* Subnautica 2
* Latest Release Package

## Installation Steps

1. Download the latest release.
2. Extract the ZIP archive.
3. Launch `Subnautica2BlueprintTracker.exe`.
4. Create a new explorer profile.
5. Start tracking your blueprint progress.

## Updating

1. Download the latest release.
2. Replace existing files.
3. Launch the application.

## Uninstallation

1. Delete the application folder.
2. Remove optional profile data if desired.
